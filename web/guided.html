    Equity in the bundled games is Maia-2 output (or an illustrative skew — see each file's _comment); regenerate with python web/build_demo.py --all --model maia2.

    Maia-2's value head reads ~41% for White at the opening rather than 50%, so the bar starts slightly below even — read it as how the game is trending for this matchup, not an absolute 50%-is-equal scale.

    Import a real game: python web/import_game.py <lichess-url>, then open ?game=imported-game.json.
